# GenPresso AI 생성 플랫폼

AI 기반 콘텐츠 생성 플랫폼. 텍스트, 이미지, 비디오 생성 및 노드 기반 워크플로우 지원.

## 🚀 주요 기능

- **AI 생성**: 텍스트/이미지/비디오 생성
- **노드 캔버스**: 드래그 앤 드롭 워크플로우
- **세그멘테이션**: 이미지 영역 분할 및 믹스
- **다크/라이트 모드**: 테마 자동 전환
- **반응형 디자인**: 모바일/태블릿/데스크톱 지원

## 📁 구조

```
├── App.tsx                        # 라우터
├── components/
│   ├── layout/                    # 메인 레이아웃
│   ├── canvas/                    # 캔버스 워크스페이스
│   ├── ui/                        # 공통 UI 컴포넌트
│   └── icons/                     # 아이콘 시스템
├── styles/globals.css             # Tailwind v4 스타일
└── guidelines/Guidelines.md       # 개발 가이드라인
```

## 🎨 디자인 시스템

**색상**
- Primary: `#4fa8d8` (브랜드 블루)
- Background: 라이트 `#ffffff` / 다크 `#141415`

**글래스모피즘**
- 투명 배경 + 12px 백드롭 블러
- CSS 변수 기반 라이트/다크모드 대응

**타이포그래피**
- 폰트: Pretendard
- 최소 크기: 12px (`text-xs`)
